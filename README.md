# 🎙️ Speech-to-Text Web Application

An AI-powered full-stack web application that converts uploaded audio files into text using the **Deepgram Speech-to-Text API**.
The application supports user authentication, transcription history management, and cloud deployment.

## 📌 Features:-
* 🔐 User Authentication (Signup / Login)
* 🎧 Upload `.mp3` and `.wav` audio files
* 🧠 AI-powered Speech-to-Text conversion using **Deepgram**
* 📜 Transcription history storage
* 🗑️ Clear transcription history
* ☁️ Fully deployed frontend and backend
* 📱 Responsive UI using Tailwind CSS

## 🛠️ Tech Stack

### Frontend:-
* React.js
* Vite
* Tailwind CSS
* Axios

### Backend:-
* Node.js
* Express.js
* Multer

### Database & Authentication:-
* MongoDB Atlas
* Supabase Authentication

### AI Integration:-
* Deepgram Speech-to-Text API

### Deployment:-
* Netlify (Frontend)
* Render (Backend)

## ⚙️ How It Works:-
1. User signs up or logs in
2. User uploads an audio file
3. Frontend sends file to backend
4. Backend sends audio to Deepgram API
5. Deepgram converts speech into text
6. Transcription is stored in MongoDB
7. Result and history are displayed on frontend

## 📌 Conclusion

The **Speech-to-Text Web Application** successfully demonstrates the integration of modern full-stack development with AI-powered speech recognition technology.
The project provides a secure and user-friendly platform where users can upload audio files and convert speech into text efficiently using the **Deepgram API**.
Through this project, important concepts such as **frontend development, backend API creation, authentication, database management, file handling, and cloud deployment** were implemented and understood practically.
The application also includes features like user authentication, transcription history management, and responsive UI design, making it a complete real-world web application.

Overall, this project enhanced practical knowledge of the **MERN ecosystem, API integration, and deployment workflows**, while showcasing how AI can be used to solve real-world problems effectively.
